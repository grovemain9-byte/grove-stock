## 戦略の核心（変更禁止）
- MA25は**日足ベース**（30分足ではない）
- 乖離率 = (終値 - 日足MA25) / 日足MA25 × 100
- コンセンサス 3/5以上 → BUY
- 損切り -5% / 利確: MA25に回帰（乖離率 >= 0） / 最大保有5営業日

## 5プレイヤー（全員ルールベース・LLM不要）

| # | 条件 |
|---|------|
| P1 | MA25乖離率 ≤ セクター閾値 |
| P2 | RSI(14) < 35 |
| P3 | close < ボリンジャーバンド下限(25,2) |
| P4 | 日経225当日騰落 > -2% |
| P5 | 直近3日出来高が減少傾向 |
